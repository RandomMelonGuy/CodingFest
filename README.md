# CodingFest
 Репозитроий, созданный командой Tobkvant к участию в конкурсе Coding Fest 2025

## УСТАНОВКА

1) Создайте виртуальную среду python в корне проекта и ак
2) Установите зависимости с помощью `pip install -r requirements.txt`
3) Запустите программу с помощью `pythonw start.pyw`

## Функциональность

### При старте приложения программа ищет файл в папке data/ и если не находит создаёт materials.xlsx

### UI делится на 4 вкладки:
+ Добавление материала
+ Просмотр и фильтрация
+ Пополнение / Списание
+ Экспорт данных

## Добавление материала
В данной вкладке расположены 7 полей ввода.
Каждое проверяется по следующим параметрам:
1) Остаток должет быть неотрицательным числом
2) Размер катушки / вес должен поступать в формате ВЕС / ДИАМЕТР (Например "100г / 120мм")
3) Сечение должно быть числом

## Просмотр и фильтрация
В данной вкладке расположена таблица со всеми используемыми данными.
Над таблицей расположены поля ввода для фильтров: ID, Вид матриала, Цвет и Статус
Фильтрация срабатывает только на полное совпадение с фильтром.
### Чтобы просмотреть даныне строки в отдельном окне нажмите ЛКМ по интересующей строке

## Пополнение / Списание
Для начала работы необходимо выбрать материал из выпадающего списка
Для списания или пополнения необходимо ввести количество в поле ввода и нажать на соответствующую кнопку

## Экспорт данных
При нажатии на кнопку открывает файловый диалог для сохранения копии рабочего файла

### Технологические данные
#### Архитектура - MVP (Mdoel View Presenter)
#### Графика - CustomTkinter, CTkTable, CTkMessagebox
#### Логика - pandas, openpyxl
#### Парадигма - ООП
